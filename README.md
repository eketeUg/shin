# SHIN: The Struggle

A 2D PvP fighting game with Igbo folklore style.

## Tech Stack
- **Frontend**: Next.js, PhaserJS, TailwindCSS
- **Backend**: NestJS, Socket.io
- **Language**: TypeScript

## Getting Started

### Prerequisites
- Node.js
- npm

### Installation

1. Install dependencies for server:
   ```bash
   cd server
   npm install
   ```

2. Install dependencies for client:
   ```bash
   cd client
   npm install
   ```

### Running the App

1. Start the server (port 3001):
   ```bash
   cd server
   npm run start:dev
   ```

2. Start the client (port 3000):
   ```bash
   cd client
   npm run dev
   ```

## Gameplay
- **Menu**: Click to start.
- **Game**: Uses Phaser for rendering.
